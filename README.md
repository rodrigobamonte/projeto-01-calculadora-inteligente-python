 🧮 Projeto 01 - Calculadora Inteligente em Python
## Sobre o Projeto
Este projeto foi desenvolvido durante meus estudos de Python com o objetivo de praticar os fundamentos da linguagem por meio da construção de uma calculadora inteligente.
A aplicação realiza operações matemáticas básicas, valida entradas e utiliza estruturas condicionais para executar diferentes cálculos de forma automatizada.
Este foi meu primeiro projeto prático utilizando Python.

## Objetivos do Projeto

- Praticar lógica de programação.
- Desenvolver uma aplicação utilizando Python.
- Aplicar estruturas condicionais.
- Construir meu primeiro projeto para portfólio.

- ## Funcionalidades

A calculadora realiza as seguintes operações:

- Soma
- Subtração
- Multiplicação
- Divisão
- Potenciação
- Raiz Quadrada
- Porcentagem

- ## Validações Implementadas

Durante o desenvolvimento foram implementadas validações como:

- Divisão por zero.
- Operação inválida.
- Identificação de resultados positivos, negativos e iguais a zero.

- ## Conceitos de Python Aplicados

Neste projeto foram utilizados os seguintes conceitos:

- Variáveis
- Operadores matemáticos
- Operadores relacionais
- Estruturas condicionais (`if`, `elif` e `else`)
- Entrada de dados (`input`)
- Conversão de tipos
- Formatação utilizando `f-strings`

- ## 🛠️ Tecnologias Utilizadas

- Python
- Google Colab
- GitHub

Este projeto foi desenvolvido exclusivamente para fins de estudo e prática de programação.
